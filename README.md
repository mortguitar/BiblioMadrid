# BiblioMadrid
### Es una aplicación para observar y calcular el tiempo de la ruta desde tu posición a una biblioteca que desearias ir con ayuda del mapeado de Android y diferentes funciones de geolocalización



Realizada con **Android Studio**, está aplicación te ayuda a **aplicar una configuracion de ubicación** para así realizar una localización precisa. Antes de utilizar la aplicación necesitas...

### Habilitar tu ubicación en Ajustes>Permisos>Permisos de Aplicaciones

También necesitarás iniciar el servidor de Spring Boot. Necesitas introducir este código en la terminal de Gradle...
Desde el propio IDE, ejecutando`./gradlew bootRun` (o bien `gradlew bootRun` si estamos en Windows)
Utilizando el **`jar`** que podemos generar con el comando `./gradlew jar build` (`gradlew jar build en Windows`) y ejecutarlo con el comando java -jar.

**El .jar generado lo podremos encontrar en la carpeta `build/libs`**
